A Combinators is something that explains the relationship between the selectors.

There are four different combinators in CSS:

1.Descendant selectors (space)

2.Child selector (>)

3.Adjacent sibiling selector(+)

4.General sibiling selector (~)

Descendand selectors

The descendant selectors matches all elements that are descendant of a specified element.

Adjacent sibiling selector

The + selectors is used to select an element that is directly after another specific element.

Child selector

The child selector select all elements that are the children of a specified element.

General sibiling selector

The general sibiling selector selects all elements that are next sibilings of a specified element.
